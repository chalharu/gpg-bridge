# gpg-bridge daemon

`daemon/` はユーザー環境で動作するローカルデーモンを提供する Rust バイナリです。

## 役割

- GPG エージェント互換の IPC インターフェース（Unix ソケット / Windows 名前付きパイプ）を提供し、GPG コマンドからの署名・鍵照会要求に応答する
- サーバーとの SSE 接続を維持し、署名結果やペアリングイベントを受信する
- HTTP クライアントによるサーバー API 通信（リトライ・Bearer 認証対応）

## 現在の実装状況

- **CLI & 設定管理**: `clap` によるコマンドライン引数解析、TOML/YAML 設定ファイル、環境変数の優先度付きマージ
- **HTTP クライアント**: `reqwest` ベースのクライアント（タイムアウト、リトライ、429/5xx バックオフ対応）
- **SSE クライアント**: Server-Sent Events による常時接続（指数バックオフ再接続、ハートビートタイムアウト検知、Last-Event-ID 再送、グレースフルシャットダウン対応）
- **IPC サーバー**: Unix ソケットリスナー（互換シンボリックリンク・パーミッション管理付き）/ Windows 名前付きパイプリスナー
- **Assuan プロトコル**: 未実装（接続受付のみのスタブ状態）
